Animation Types

Hover

Animation only plays when you hover with mouse: .juflair_hover

Repeat

  • .jusaflair_rep_inf: infinity loop
  • .juflair_rep_2 : repeat 2x
  • .juflair_rep_3 : repeat 3x

Delay

  • .jusaflair_del_1 to .jusaflair_del_5: delay time options from 1s to 5s
  • .jusaflair_del_0-5 to .jusaflair_del_5-5: delay time options from 0.5s to 5.5s

Speed

  • .juflair_spe_light : speed 0.2s
  • .juflair_spe_fast : speed 0.5s
  • .juflair_spe_normal (default) : speed 1s
  • .juflair_spe_slow: speed 3s
  • .juflair_spe_slowmotion : speed 5s
